#------------------------------------------------------------------------------------------
#這裡是設定瞬移路徑及瞬移路徑種類之地圖清單。
#------------------------------------------------------------------------------------------
#格式: <目前地圖> <下一張地圖> <瞬移路徑開關> [<與傳送點距離> | <區塊範圍>]
#   <目前地圖>:		目前所在地圖, 在tables/maps.txt中可找到之地圖名稱去掉.rsw
#   <下一張地圖>:	要前往的下一張地圖, 在tables/maps.txt中可找到之地圖名稱去掉.rsw
#   <瞬移路徑開關>:	0=不使用瞬移路徑
#			1=指定距離方式瞬移
#			2=設定區塊範圍方式瞬移
#   <與傳送點距離>:	瞬移路徑開關設為 1 時, 可自訂與傳送點距離, 
#			若不設定則使用 teleRouteDist 設定的距離
#   <區塊範圍>:		<左上X軸座標> <左上Y軸座標> <右下X軸座標> <右下Y軸座標>
#			瞬移路徑開關設為 2 時, 需指定要達到的區塊範圍
#------------------------------------------------------------------------------------------
#說明:
#   1.當所在地圖為城市地圖(在tables/cities.txt中找的到)，則無作用。
#   2.此功能與偏好路徑及鎖定地圖沒有直接的關係。
#------------------------------------------------------------------------------------------
#相關變數:
#   config.txt
#	-teleRoute
#	 使用順移功能加速到達定目標地圖(0=關、1=開)
#	-teleRouteDist
#	 距離傳送點多少格以外就用瞬移方式(數值不要設太小免得飛太久)
#   timeouts.txt
#	-ai_teleRoute
#	 瞬移路徑延遲時間(秒)
#------------------------------------------------------------------------------------------
# 範例:
#prt_fild08 moc_fild01 0 <===用走的 
#prt_fild08 moc_fild01 1 150 <==飛到離傳點150格內停下來走 
#prt_fild08 moc_fild01 2 <左上X軸座標> <左上Y軸座標> <右下X軸座標> <右下Y軸座標> <==飛進區域才用走的
