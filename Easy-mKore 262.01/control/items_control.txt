#------------------------------------------------------------------------------------------
#這裡是設定要自動存放、自動賣出之物品清單。
#------------------------------------------------------------------------------------------
#格式:<物品名稱> <數量> <自動存放開關> <自動賣出開關> <手堆車物品自動存放開關>
# <物品名稱>:在tables/items.txt中可找到之物品名稱
# <數量>:要留在身上的物品數量
# <自動存放開關>:0=關、1=開
# <自動賣出開關>:0=關、1=開
#------------------------------------------------------------------------------------------
#說明:
# 1.大於此數量的物品都會被存入倉庫或是賣給NPC。
# 2.數量設為0時表示全部存入或賣出。
# 3.當<自動存放開關>及<自動賣出開關>都設為1時，
#   會先將物品存放至倉庫(除非倉庫已經滿了)，剩餘的物品則賣出給商店。
# 4.當<手堆車物品自動存放開關>設為1時，手堆車內物品會存放至倉庫(除非倉庫已經滿了)。
#------------------------------------------------------------------------------------------
#例如:
# 1.羽毛 20 0 1
# 傑勒比結晶 0 0 1
# 解說:會將羽毛賣出到只剩20個(假設原本大於20)，而傑勒比結晶則會全部賣出。
#
# 2.羽毛 20 1 1
# 傑勒比結晶 0 1 1
# 解說:會將羽毛存放至倉庫到只剩20個(假設原本大於20)，而傑勒比結晶則會全部存入倉庫。
# 若倉庫已無法存放更多物品，便會將剩下的羽毛賣出到只剩20個(假設還大於20)，
# 而剩下的傑勒比結晶則會全部賣出。
#------------------------------------------------------------------------------------------
