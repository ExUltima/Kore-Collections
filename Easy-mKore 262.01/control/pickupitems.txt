# <物品名稱>:在tables/items.txt中可找到之物品名稱

#說明:
# 1.該物品出現在視線內，會自動撿取。
# 2.物品名稱為"all"可預設所有物品是否撿取。
#------------------------------------------------------------------------------------------
#例如:
# 1.all 1
# 2.all 0
# 3.all 1 50
# 花 -1
# 解說:如果道具欄有此物品會自動丟棄,該物品數量變動時才做偵測。
# 花 0
# 解說:會撿取除了花以外的所有物品。
# 花 1
# 解說:只會撿取花，其他物品則不撿取。
# 3.重要物品
# 花 2
# 解說:會優先撿取花。
# 4.撿取物品上限
# 花 1 50
# 解說:設定撿取或重要物品後再增設撿取物品上限數量，超過上限則不在撿取，若不設定則無限制
